(FYP) AI Enhanced Cloud Storage For IKM Student

AuraDrive is an intelligent cloud storage platform I built with a Python/Flask backend and a React frontend, designed for role-based access in an educational setting. The application features a full hierarchical file system, multi-drive scopes (Personal, Community), and a "Desktop-Class" UI with previews and multi-select. The core feature is DriveAi, a Groq Llama 3-powered assistant that executes natural language commands for file management, search, and content summarization.

Core Features

  • AI-Powered File Management (DriveAi): Use natural language commands (e.g., "find all pdfs from last week and put them in a folder named 'Reports'") to manage files.
  • Hierarchical File System: A complete folder and file structure with full create, read, update, and delete (CRUD) capabilities.
  • Multi-Drive Scopes: Separate "Personal" drives for private files and a "Community" drive for shared resources among students.
  • Role-Based Access Control: Admin, teacher, and student roles with different permissions to ensure data security and privacy.
  • Desktop-Class UI: Modern interface built with React, featuring drag-and-drop, file previews, and a responsive design.
